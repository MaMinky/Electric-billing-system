## 🛠️ Cấu trúc dự án
- `Main.java`: Entry point của chương trình.
- `Calculator.java`: Logic tính toán hóa đơn 
- `CustomerManager.java`: Quản lý danh sách khách hàng.

## 🚀 Kịch bản Demo Git
1. Trưởng nhóm tạo Repo và cấu trúc cơ bản.
2. Thành viên `fork` dự án và tạo nhánh `feature/update-price`.
3. Cả 2 cùng sửa biến `BASE_PRICE` trong `Calculator.java` để tạo Conflict.
4. Trình diễn cách giải quyết xung đột thông qua Pull Request.
